So I wrote a script that goes through all of these values, subtracts 1, and prints as a char:

# goes through all characters, get the ord for it, subtracts, turns it into a char, then joins the array as a string print(''.join([chr(ord(i) - 1) for i in "jdug|tus2oht`5s4ou`i2ee4o`28c32b7:~"]))
